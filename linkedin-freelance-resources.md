## LinkedIn Jobs — Freelance Filters

Filter for contract/freelance work:

1. Go to: linkedin.com/jobs
2. Search: "AI Engineer" OR "LLM" OR "RAG" OR "Machine Learning"
3. Filter:
   - **Job Type:** Contract
   - **Experience Level:** Entry Level / Associate
   - **Location:** Remote (or your preferred city)
   - **Date Posted:** Past 24 hours / Past week

### Search Queries to Use

| Search Query | Purpose |
|-------------|---------|
| `AI Engineer freelance` | Find AI contract work |
| `LLM integration contract` | LLM API projects |
| `RAG developer remote` | RAG-specific work |
| `Generative AI freelancer` | GenAI projects |
| `Python FastAPI contract` | Backend AI work |
| `Chatbot development remote` | Chatbot projects |
| `Machine Learning contractor` | ML contracts |

### Job Alerts Setup

1. Save each search as a Job Alert
2. LinkedIn will email you when new matches post
3. Set alerts for 5+ queries to maximize opportunities
